================================================================================
    MICROSOFT 基础类库 : Harmonia启动配置 项目概述
===============================================================================

应用程序向导已为您创建了此 Harmonia启动配置 应用程序。此应用程序不仅演示 Microsoft 基础类的基本使用方法，还可作为您编写应用程序的起点。

本文件概要介绍组成 Harmonia启动配置 应用程序的每个文件的内容。

Harmonia启动配置.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

Harmonia启动配置.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

Harmonia启动配置.h
    这是应用程序的主头文件。
    其中包括其他项目特定的标头（包括 Resource.h），并声明 CHarmonia启动配置App 应用程序类。

Harmonia启动配置.cpp
    这是包含应用程序类 CHarmonia启动配置App 的主应用程序源文件。

Harmonia启动配置.rc
    这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。项目资源包含在 2052 中。

res\Harmonia启动配置.ico
    这是用作应用程序图标的图标文件。此图标包括在主资源文件 Harmonia启动配置.rc 中。

res\Harmonia.rc2
    此文件包含不在 Microsoft Visual C++ 中进行编辑的资源。您应该将不可由资源编辑器编辑的所有资源放在此文件中。


/////////////////////////////////////////////////////////////////////////////

应用程序向导创建一个对话框类：

Harmonia启动配置Dlg.h、Harmonia启动配置Dlg.cpp - 对话框
    这些文件包含 CHarmonia启动配置Dlg 类。此类定义应用程序的主对话框的行为。对话框模板包含在 Harmonia启动配置.rc 中，该文件可以在 Microsoft Visual C++ 中编辑。

/////////////////////////////////////////////////////////////////////////////

其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 Harmonia启动配置.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
    这是标准头文件，可用于定义新的资源 ID。Microsoft Visual C++ 将读取并更新此文件。
